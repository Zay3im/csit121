What is WebAuthn?

WebAuthn stands for Web Authentication. It is a web standard that lets websites use safer ways to log people in. Instead of only using a password, a site can let someone use a passkey, Face ID, Touch ID, Windows Hello, or a security key.

The main thing WebAuthn does is help prove that the person logging in is really the owner of the account. It does this with public key credentials. That means the website and device use matching keys to verify the login.

I picked this topic because it connects to passwords and online security. A lot of people use weak passwords or reuse the same ones, so this standard is important. It shows how websites are starting to move toward safer sign ins.
