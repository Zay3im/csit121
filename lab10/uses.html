Examples of WebAuthn

WebAuthn is used when a website lets someone sign in with a passkey. A person might use Face ID, Touch ID, a PIN, or Windows Hello. This allows them to log in without typing a password every time.

It can also be used with security keys. A security key is a small device that someone plugs in or taps to prove their identity. Some people use these for accounts that need extra protection.

A example would be a website with a sign in with passkey button. The browser asks the device to confirm the user before logging in. This shows how the website, browser, and device work together.
